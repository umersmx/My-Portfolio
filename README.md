## 🌟 About This Project

This repository contains the source code for my personal portfolio website. Transitioning into the world of **Full-Stack Development**, I built this site from the ground up to reflect my technical abilities, my educational background, and the modern UI/UX design standards I strive to implement in my applications.

The design utilizes sleek **glassmorphism** effects, **CSS Grid/Flexbox** architectures, and smooth scroll-triggered animations using the Intersection Observer API.

---

## 🚀 Key Features

- **Modern Glassmorphic UI:** A visually striking aesthetic using vibrant gradients and frosted glass elements.
- **Scroll-Triggered Animations:** Elements naturally slide and fade into view as the user navigates down the page.
- **Responsive Layout:** Perfectly scales across mobile devices, tablets, and large desktop monitors.
- **Dynamic Typing Effect:** Custom JavaScript logic powers a typing animation in the hero section.
- **Interactive Project Showcase:** Highlighting my best work across MERN stack, Python, and C++.

---

## 🛠️ Built With

Although my focus is on full-stack technologies like React and Node.js, this specific portfolio is deliberately built with vanilla web technologies to demonstrate core foundational knowledge:

- **HTML5** (Semantic structure & SEO metadata)
- **CSS3** (Custom properties, Flexbox, Grid, Animations, Media Queries)
- **JavaScript (ES6+)** (DOM Manipulation, Intersection Observer API)
